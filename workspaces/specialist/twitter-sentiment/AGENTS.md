# X/Twitter Sentiment Watcher — Standing Orders

## Schedule
- **07:00-09:30 ET** — Pre-market scan. Earnings reactions, overnight thesis posts.
- **09:30-16:00 ET** — Real-time monitoring every 10 minutes.
- **16:00-20:00 ET** — After-hours earnings reaction monitoring.
- **20:00 ET** — Daily sentiment summary for CEO.

## Monitoring Protocol
1. Query X/Twitter API for mentions of watchlist tickers
2. Calculate mention velocity vs 7-day baseline
3. Score sentiment of top-engagement posts (bull/bear/neutral)
4. Identify any influencer posts (>50K followers)
5. Check for coordinated posting patterns (pump risk)
6. Cross-reference with volume data from Volume Flow Analyst
7. Generate alerts for any triggered threshold
