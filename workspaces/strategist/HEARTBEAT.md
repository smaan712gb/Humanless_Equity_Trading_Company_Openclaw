# Strategist Heartbeat Checklist (Every 5 minutes)

- [ ] Check for new Analyst reports (edge score >= 6 required)
- [ ] Review Position Manager's latest position update
- [ ] Check Volume Flow alerts — any accumulation/distribution signals?
- [ ] Check Technical Analyst — any breakout/breakdown alerts?
- [ ] Check Options Flow — any unusual activity on watched tickers?
- [ ] Check Futures Watcher — NQ/ES directional bias
- [ ] If setup identified: calculate Kelly size, generate Trade Plan, send to Compliance
- [ ] Update rolling Kelly parameters if any trades completed
- [ ] If 3+ consecutive losses, pause and review edge assumptions
