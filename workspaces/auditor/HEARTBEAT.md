# Auditor Heartbeat Checklist (Daily at 20:00 ET)

- [ ] Retrieve all session transcripts from today's trading
- [ ] Run Attacker phase: find every error, missed signal, rule violation
- [ ] Run Defender phase: check each agent's reasoning at the time
- [ ] Run Judge phase: severity + confidence for each finding
- [ ] Critical findings → escalate immediately to CEO / board
- [ ] If behavioral drift detected (agent acting against SOUL.md) → flag
- [ ] If RSI candidate identified → prepare backtest proposal
- [ ] Generate daily audit report → send to CEO
