# CEO Heartbeat Checklist

- [ ] Check daily P&L vs $20K target — request portfolio summary from Position Manager
- [ ] Review any pending approval requests (new hires, strategy changes)
- [ ] Check for Compliance alerts or vetoes
- [ ] Check for Auditor critical findings
- [ ] Review Scout's latest ticker discoveries — approve or reject
- [ ] Check agent health — are all agents reporting?
- [ ] If daily P&L > $40K, consider scaling down
- [ ] If drawdown approaching -2%, escalate to halt new entries
- [ ] Log status update to daily diary
