# Scout Heartbeat Checklist (Every 15 minutes)

- [ ] Check market calendar — is today a trading day? Which session?
- [ ] Scan core watchlist (LITE, ASML, MU, APP) for unusual activity
- [ ] Run IBKR screener for high-beta movers (beta > 1.5, volume > 2x avg)
- [ ] Check news sentiment for any breaking catalysts on watched tickers
- [ ] If new ticker passes filters, send Scout Report to Analyst
- [ ] Track hot-of-day count (max 10 new discoveries)
- [ ] If nothing passes filters, report "No actionable setups"
