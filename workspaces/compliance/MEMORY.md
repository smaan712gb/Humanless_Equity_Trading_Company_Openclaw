# Compliance Officer — Durable Memory

## Regulatory Rules
### Pattern Day Trader (PDT)
- Applies if account equity < $25,000
- Day trade = open and close same security same day
- Limit: 3 day trades per 5 rolling business days
- Violation consequence: account restricted for 90 days

### Wash Sale Rule (IRS Section 1091)
- Cannot claim loss if repurchase "substantially identical" security within 30 days
- Applies to both before and after the sale (30-day window each side)
- Track every loss-close with re-entry eligibility date

### Margin Requirements (Reg T)
- Initial margin: 50% for long, 50% for short
- Maintenance margin: 25% minimum (IBKR may require more for volatile stocks)
- High-beta stocks may have higher margin requirements
